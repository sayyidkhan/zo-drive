# Changelog

All notable Zo Drive changes are recorded here. GUI and CLI releases are
versioned independently from 20 July 2026 onward.

## Unreleased

### Changed

- Added release and changelog guidance for choosing GUI and CLI version
  increments.

## GUI v0.2.0 - 2026-07-20

### Added

- Independent GUI release track and visible GUI version in the browser guide.
- GUI release documentation and `gui-v0.2.0` Git tag.

### Changed

- Moved landing-page navigation from the Drive header into the account menu;
  the Zo Drive brand remains a link to the landing page.

## CLI v0.1.3 - 2026-07-20

### Added

- Independent CLI release track and `cli-v0.1.3` Git tag.
- `zo-drive --version`, available before cloud configuration.

### Changed

- CLI installation, local-to-Zo connection, and update documentation now use
  CLI-specific release tags.

## Legacy unified releases

Before independent tracks, the repository used one version for all packages.

### v0.1.1 - 2026-07-20
